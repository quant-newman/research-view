## 诊断一:分源相关性与独立自由度(2026-07-04 已跑,等使用者审)

方法:每日对 47 节点做各源截面 z(与 evidence._z 同口径:总体 std、截断±3),
z 面板按(日期×节点)展开成长向量,源两两 Pearson 相关;
有效自由度 Neff = (Σλ)²/Σλ²(相关阵特征值)与 N/(1+(N−1)·avg|corr|) 两种口径。

### 全史段 20240701→20260703(482 开市日,22,654 节点日观测,3 全史源)

|  | price | mf | lhb |
|---|---|---|---|
| price(5日等权涨幅) | 1 | **0.296** | 0.098 |
| mf(5日累计主力净额) | 0.296 | 1 | 0.045 |
| lhb(当日龙虎榜净买) | 0.098 | 0.045 | 1 |

avg\|corr\| = 0.146,**Neff = 2.81/3(特征值法)、2.32/3(均相关法)**。

### 短窗段 20260601→20260703(24 开市日,1,128 观测,+研报/新闻)

price-mf 0.328(与全史一致);research 与其余源近正交(对 mf 甚至 −0.092);
news 对各源 |corr|≤0.09。**Neff = 4.73/5(特征值法)、3.77/5(均相关法)**。

### 读法(不含任何参数结论,等使用者)

- 外部审查担心"六源恐只有 2-3 个独立自由度":**实测不是最坏情形**——3 个全史源有约 2.8 个
  有效自由度,唯一明显重叠是行情×资金(0.30,主力买推涨本就同步);龙虎榜与两者基本独立。
- 短窗看,研报/新闻与价量源近正交(信息增量存在),但样本太短(见免责),只算方向性参考。
- 方法学注意:lhb 稀疏(多数节点日=0),零值会**低估**它与其他源的相关;news 仅 3 个交易日,
  该行该列**统计无效,只列不采信**。

## 三个诊断的合并读法(诚实版)

1. **共振分的 3 源历史骨架(行情+资金+龙虎榜,生产权重)在过去两年没有截面预测力**:
   IC≈0 且所有 horizon CI 含 0,分位不单调,Q5−Q1 与随机安慰剂无法区分。
   幸存者偏差本应让"今日赢家池"的动量信号偏乐观——即便有这个顺风,结果仍是零,**空结果更可信**。
2. **龙虎榜是唯一有历史信息量的源**(h1 IC +0.043 显著,衰减平缓到 h10 仍≈+0.02),
   但它在生产权重里恰是三者最低(0.6 vs 行情/资金各 1.0)。
3. **horizon 无明显峰**:5 日不特殊也不更差,单看历史没有改 horizon 的依据。
4. **对在线系统的正确解读**:历史台架测的是"六源退化成 3 源、没有新闻/研报/信函、没有 LLM
   研判层(方向可不同意共振)、没有 B6→B8 漏斗"的裸骨架。骨架无 alpha 意味着:
   在线系统的增量假设全部压在**新闻/研报/LLM 综合层**上——这只能靠 B7 在线样本检验,
   也再次确认 DECISIONS #22 冻结样本的珍贵性。B7 命中率的合理零假设=掷硬币,别抱 60%+ 幻想。
